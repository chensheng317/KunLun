# KunLun (昆仑工坊) UI/UX 全局配色系统与开发指南

## 1. 项目背景与设计基调 (Context & Vibe)
- **项目定位**：昆仑工坊——基于 AutoGLM 的小微电商全域数字中枢。
- **视觉风格**：青黑撞色、极简、赛博工业、科技感、中心化控制感。
- **UI 目标**：拒绝廉价感，打造具备商业级质感的“数字控制中枢”。UI 需要给人一种科技压制感。

## 2. 全局颜色变量定义 (Color Variables)
请在开发中严格使用以下颜色代码（建议配置为 Tailwind CSS 主题变量或 CSS 变量）。

### 2.1 基础层级 (Base Layout)
- **`bg-screen` (全局页面背景)**: `#161823` (漆黑) - 深沉的基底。
- **`bg-surface` (卡片/模块/容器背景)**: `#2D2B38` (青灰色) - 用于承载信息，与主背景形成第一层深度。
- **`bg-surface-alt` (备用/次级容器背景)**: `#053154` (暗宝石) - 用于图表背景、特殊侧边栏或需要更深邃质感的区块。

### 2.2 品牌与交互高亮 (Brand & Accent)
- **`accent-primary` (核心品牌色/高亮)**: `#3eede7` (碧蓝) - 视觉核心！用于主按钮、核心指标数值、进度条、激活状态、赛博辉光 (Glow)。
- **`accent-secondary` (辅助强调色)**: `#5EB8AC` (铜绿色) - 用于次要按钮、图表渐变色过渡、副标题高亮、在线/健康状态指示灯。

### 2.3 文本系统 (Typography)
- **`text-primary` (主文本)**: `#E0E6ED` - 极浅的蓝灰色，确保在暗色背景下的绝对可读性。
- **`text-secondary` (次文本)**: `#7D8A99` - 用于备注、表格表头、时间戳、占位符 (Placeholder)。
- **`text-inverse` (反色文本)**: `#161823` - 仅用于当背景色为 `accent-primary` (#3eede7) 时（例如主按钮上的文字）。

### 2.4 边框与分割线 (Borders & Dividers)
- **`border-base` (基础边框)**: `#3A3F58` - 用于模块分割、极细的卡片描边。
- **`border-glow` (发光边框)**: `rgba(62, 237, 231, 0.4)` - 配合 `accent-primary` 使用，用于当前激活/选中的模块，如“正在执行任务的设备卡片”。

## 3. 核心组件开发规范 (Component Guidelines)

在生成 React 组件时，请遵循以下样式映射：

1. **Dashboard 大屏卡片 (Cards)**:
   - 背景使用 `bg-surface`。
   - 增加极细的内边框 `1px solid rgba(255, 255, 255, 0.05)` 增加实体感。
   - 边角统一使用圆角 `rounded-xl` 或 `rounded-2xl`。

2. **AI 自然语言输入框 (LLM Command Input)**:
   - 默认状态：背景 `bg-surface`，边框 `border-base`。
   - Hover/Focus 状态：边框变为 `accent-primary` (#3eede7)，并带有一层轻微的 `box-shadow: 0 0 10px rgba(62, 237, 231, 0.2)` 辉光效果。

3. **数据看板 (Data Visualization / Echarts)**:
   - 柱状图/折线图的主色调必须采用 `#3eede7` 到 `#5EB8AC` 的线性渐变。
   - 隐藏多余的网格线，网格线颜色若保留则用 `#2D2B38`。

4. **状态指示灯 (Status Indicators)**:
   - “设备在线/任务运行中”：使用 `#3eede7` 并添加呼吸灯动画 (Pulse animation)。
   - “设备空闲/合规就绪”：使用 `#5EB8AC`。
   - “设备离线”：使用 `#7D8A99`。

## 4. Tailwind CSS 配置建议 (请应用此配置)

// tailwind.config.js 示例扩展
module.exports = {
  theme: {
    extend: {
      colors: {
        'nexus-bg': '#161823',
        'nexus-surface': '#2D2B38',
        'nexus-surface-alt': '#053154',
        'nexus-primary': '#3eede7',
        'nexus-secondary': '#5EB8AC',
        'nexus-text': '#E0E6ED',
        'nexus-muted': '#7D8A99',
        'nexus-border': '#3A3F58',
      },
      boxShadow: {
        'cyber-glow': '0 0 15px rgba(62, 237, 231, 0.3)',
      }
    }
  }
}
